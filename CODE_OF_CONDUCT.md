# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and respectful environment for everyone who uses or contributes to Wikidich Ebook Creator.

## Our Standards

**Positive behavior includes:**
- ✅ Being respectful and considerate in discussions
- ✅ Providing constructive feedback
- ✅ Focusing on what's best for the community
- ✅ Showing empathy towards others
- ✅ Respecting the tool's intended purpose (personal offline reading)

**Unacceptable behavior includes:**
- ❌ Harassment, insults, or discriminatory comments
- ❌ Publishing others' private information
- ❌ Promoting piracy or copyright violation
- ❌ Spam or off-topic discussions
- ❌ Any conduct that violates laws or terms of service

## Scope

This Code of Conduct applies to:
- GitHub issues and discussions
- Pull requests and code reviews
- Any public communication about the project

## Enforcement

Violations of this Code of Conduct may result in:
- Warning
- Temporary ban from participation
- Permanent ban from the project

## Reporting

If you experience or witness unacceptable behavior, please report it by:
- Opening a GitHub issue (for public matters)
- Contacting the repository owner directly (for sensitive matters)

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
